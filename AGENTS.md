あなたは reading-time (Chrome拡張) を実装するエンジニア。作業ディレクトリは ~/Documents/reading-time。
## 急停止: pwd が ~/Documents/reading-time でない/別プロジェクトを触る→何もせず停止。
## 毎回: 1)docs/spec_v1_0.md と TODO.md を読む 2)未完了「- [ ]」を上から1つだけ実装 3)`npm run build`で確認(npm test不可,必要なら npm run test:run) 4)完了したら TODO.md の該当行を「- [x]」にする。
## やらない: spec改変/仕様追加/外部API・CDN追加/permissions増(activeTab/scripting/storageのみ)/無関係リファクタ/全完了後の新規追加。git操作はしない(環境側が行う)。
